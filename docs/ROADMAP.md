# Roadmap

## Fase 1 — Modello dati
- Definire unità di misura standard (mm, cm, kN).
- Digitalizzare tabelle e coefficienti storici.
- Normalizzare nomi e riferimenti ai capitoli.

## Fase 2 — Motore di calcolo
- Calcolo proprietà di sezione.
- Interpolazioni tabellari (lineare e bilineare).
- Implementazione procedure RD 2229/1939.

## Fase 3 — GUI
- Input guidato per materiali, sezione, armature.
- Visualizzazione step-by-step del calcolo.
- Esportazione report (PDF).
